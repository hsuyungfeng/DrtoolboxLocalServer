## Key Decisions Made (Phase 1 Context)

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Model + Quantization | Qwen 3.6 + Q8_0 | ~250-300ms latency, ~10GB VRAM, avoids OOM |
| Output Generation | Streaming (token-by-token) | Better perceived latency, fits <3s target |
| Concurrency | Dynamic batching (1-4) | Flexible scaling, prevents guaranteed VRAM overrun |
| Search Method | Semantic-only (Chroma) | Excellent conceptual matching, existing infrastructure |
| Chunking | Small (256-512 tokens) | Precise retrieval, easier citation tracking |
| Vector DB | Keep existing Chroma | Fast Phase 1 start, no migration overhead |
| Doc Format | Library-based (PyPDF2, python-docx) | Simple, no cloud dependency |
| API Framework | Flask | Lightweight, adequate for clinic scale |
| Citation | Full document + section tracking | Audit trail, medical accountability |
| Memory Mgmt | nvidia-smi polling + token limit + daily restart | Safe 24h operation |

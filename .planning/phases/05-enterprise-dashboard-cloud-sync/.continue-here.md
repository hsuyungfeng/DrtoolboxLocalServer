## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| QueryAnswer without llm_server | QueryAnswer.query_with_llm() requires llm_server parameter; initial implementation didn't pass it | advisory | Always check method signatures in rag/query.py; use semantic search as fallback for testing |
| SearchResult object access | Chroma SearchResult objects use attributes, not dict get(); initial code used dict patterns | advisory | Use getattr() for SearchResult attributes; test with actual Chroma results early |
| Git lock files during commits | .git/index.lock persists after failed commits; blocks subsequent operations | advisory | Remove lock file before retrying: `rm -f .git/index.lock && sleep 1 && git ...` |

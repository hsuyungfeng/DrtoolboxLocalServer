# Drtoolbox Local Server (診所本地 AI 訓練與營運系統)

Drtoolbox 是一個專為診所設計的**隱私優先 (Privacy-First)** 本地 AI 系統。它不僅能精準回答醫療諮詢，更是一套完整的「高品質數據生產線」，能自動將診所的文件轉化為 AI 訓練素材，並提供經營決策洞察。

---

## 🚀 專案目前狀態 (Project Status)
本專案已進入 **「全自動化成長階段」**。系統已實現從「資料餵食」到「AI 自動校正」再到「官網知識庫同步」的閉環流程：
- **核心推理**：採用本地 Qwen-35B 等級模型，具備 PageIndex 深度推理能力。
- **數據閉環**：AI 能自我評估信心，高信心回覆自動存入訓練集。
- **營運賦能**：具備每週 CRM 洞察報表與高風險對話即時監控。

---

## 🌟 核心功能描述 (Key Functions)

### 1. 🧠 智慧自我評估與自動收錄 (Self-Scoring & Auto-Curation)
- **數據依賴度評分**：每一筆回覆都會顯示 **「🎯 數據依賴度 %」**，代表該回答有多少比例是基於您的本地文件，而非 AI 的通用知識。
- **高信心自動歸檔**：當信心分數 **≥ 85%** 時，系統會視為「黃金數據」自動存入 `verified_training_data.jsonl`。
- **低信心事實核查**：信心分數較低或涉及風險的對話，會自動排程至夜間進行「網實事実核查 (Web Fact-Check)」。

### 2. 📊 高效率批量校正介面 (Batch Curation Workflow)
- **批量處理**：支援「全選」與「核取方塊」，醫師可一次審核數十筆模擬或真實對話。
- **即時編輯快取**：在勾選狀態下，對提問或回答的任何修改都會自動暫存，最後點擊 **「批量儲存」** 即可一次完成所有校正。
- **紫色/藍色標籤機制**：區分「主動模擬 (Proactive)」與「Hermes 建議修正 (Draft)」，加速審核決策。

### 3. 🚨 高風險醫療安全監控 (Staff Escalation Alerts)
- **即時警示**：自動偵測「劇烈疼痛、化膿、視力模糊」等 15 個核心紅旗徵兆。
- **視覺脈衝**：偵測到風險時，對話框會出現 **紅色閃爍警報** 並標註「醫師請立即人工接管」。

### 4. 📝 官網文章自動化同步 (Article Sync & Auto-Post)
- **散文轉換**：將 QA 對話自動改寫為專業醫學文章。
- **自動上傳**：透過 Playwright 腳本，一鍵將數十篇文章自動填入並存入 `doctor-toolbox.com` 後台。

## 📁 檔案結構用途 (File Structure Roles)

### 🏗️ 核心系統 (`/src`)
- `agent/hermes_core.py`: **系統大腦**。負責任務路由、高風險偵測、信心評估與串流控制。
- `rag_engine.py`: **混合檢索引擎**。整合 SQL 資料庫、PageIndex 語義樹與 N-gram 快速索引。
- `data_loader.py`: **多模態處理器**。處理 PDF、PPTX、影音與圖片 OCR 的底層解析。
- `api/routes/`: **通訊接口**。定義了與 Dashboard 和 Chat 互動的所有後端介面。

### ⚙️ 自動化排程 (`/scripts`)
- `nightly_qa_generator.py`: 每天凌晨掃描文件，自動生成模擬病患問答。
- `nightly_fact_check.py`: 自動對低信心對話進行網路搜尋驗證，並擬定修正草案。
- `weekly_crm_insights.py`: 每週一生成營運報表，分析療程熱度與病患痛點。
- `auto_post_to_toolbox.py`: 官網後台自動化同步腳本。

### 💾 數據與知識庫 (`/data`)
- `documents/special/`: 存放診所私有的療程、行銷與操作手冊。
- `documents/general/`: 存放一般醫學衛教知識（如糖尿病、內科手冊）。
- `pageindex/`: 預先計算好的文獻語義邏輯樹，用於深度推理。
- `verified_training_data.jsonl`: **最終產出**。存放醫師批准的高品質訓練對，用於未來模型微調。

---

## 🛡️ 核心運作準則
1. **嚴禁報價**：絕對不輸出金錢數字。
2. **本地優先**：所有病患隱私與原始文件絕不上雲端。
3. **繁體中文一致性**：口吻親切、專業、權威。
4. **人工最終審核**：AI 提供建議，醫療決策權始終在醫師手中。

## 🛠️ 開發與維護指令
- **啟動伺服器**：`uv run python src/api/app.py`
- **執行手動核查**：在控制面板點擊「🔍 執行今日網實核查」。
- **手動同步官網**：`uv run python scripts/auto_post_to_toolbox.py`

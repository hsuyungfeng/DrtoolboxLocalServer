# 各平台文案規範

## Instagram 貼文

### 語氣
- 輕鬆親切，像跟好朋友分享
- 口語化，避免過於生硬的醫療用語
- 適度使用 emoji 增加視覺吸引力

### 格式
- 字數：150-300字
- 短句分行，每行不超過15字
- 開頭用【】框住吸睛標題
- 結尾加 hashtag（5-10個）
- 包含 CTA（私訊預約、點連結等）

### Hashtag 常用標籤
- #緻妍外科 #台中醫美 #太平醫美
- 依療程加入：#非侵入式 #臉部拉提 #抗老 #膠原蛋白 #皺紋 等
- 季節/節日相關標籤

### 範例語氣
- ✅「姐妹們～你有沒有覺得最近照鏡子，臉好像偷偷垂下來了？」
- ❌「本診所引進最新非侵入式臉部電磁緊緻儀器」

## Facebook 貼文

### 語氣
- 專業溫暖，知識型內容
- 可使用療程原理說明，但要白話易懂
- 用數據佐證效果（如「膠原蛋白增加26%」）
- 適度使用 emoji，但比 IG 少

### 格式
- 字數：300-600字
- 段落分明，使用小標題
- 開頭用情境或痛點引起共鳴
- 中段詳細說明療程
- 結尾放優惠資訊 + 預約方式

### 範例語氣
- ✅「隨著年齡增長，臉部肌肉逐漸流失支撐力，導致皮膚鬆弛下垂。EMFACE 運用 RF 射頻 + HIFES 高強度聚焦電磁刺激雙重技術，從根本強化臉部肌肉框架。」
- ❌「快來做臉部拉提喔～超讚的！」

## LINE 官方帳號貼文

### 語氣
- 親切直接，像一對一私訊推薦
- 口語化但保持專業感
- emoji 適量使用，增加可讀性

### 格式
- 字數：200-400字
- 用 ━━━ 分隔線劃分區塊（痛點 / 療程介紹 / 數據 / 體驗 / 聯絡）
- 開頭用 🌟 或 ✨ 搭配標題吸引注意
- 痛點用 😟 列出 2-3 點引起共鳴
- 數據用 ✅ 條列，一行一項
- 結尾放診所資訊 + CTA（「點選預約諮詢」）
- 建議搭配圖文訊息（多張圖片輪播）

### 圖片搭配建議
- 封面圖：產品主視覺或療程示意圖
- 中間圖：實際療程照片
- 結尾圖：效果數據圖或 before/after

### 範例語氣
- ✅「姐妹們有這些困擾嗎？😟 法令紋越來越明顯」
- ❌「本診所最新引進之臉部電磁緊緻儀器現正優惠中」

## TikTok 動態圖文影片

### 風格
- 捲動式資訊圖（文字由下往上滾動，背景圖片漸變切換）
- 9:16 直式畫面
- 15-30秒快節奏

### 影片結構（5場景模板）
1. **痛點場景**（4秒）— 一句話點出問題，引起共鳴
2. **產品介紹**（4秒）— 療程名稱 + 核心技術一句話
3. **數據展示**（5秒）— 四格動態數字跳動
4. **療程體驗**（5秒）— 4個特點逐一滑入
5. **診所資訊**（4秒）— 名稱 + 地址 + 電話

### 動畫效果
- Ken Burns 背景緩慢縮放
- 文字淡入 + 由下往上滾動
- 數字跳動動畫（從 0 跳到目標值）
- 體驗特點逐一從左滑入
- 頂部進度條（類似 IG Story）

### 技術實作
- 使用 web-artifacts-builder 建立 React + TypeScript 專案
- 打包為單一 HTML 或用 dev server 預覽
- 產品圖片放 `public/` 目錄

### 文字規範
- 每場景文字精簡，標題不超過 10 字
- 副標題不超過 20 字
- 字體大且醒目，需加文字陰影確保可讀性

## 通用規範

- 所有文案使用繁體中文
- 不誇大療效，使用「幫助改善」、「有助於」等措辭
- 價格需使用者確認才能寫入
- 優惠活動標註有效期間
- 結尾必須包含診所聯絡資訊
